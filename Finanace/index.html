pystock

This is a small Python library which has an easy to use API for portfolio optimization using models like Capital Asset Pricing Model (CAPM), Single Index Model (SIM), Fama French Three and Five Factor Model (FF3, FF5) and more. The library is available on PyPI and can be installed using pip. The source code is available on GitHub and can be found by link below.

optionalyzer

This is a library which can be used to build option strategies. The library implements the Black-Scholes model for option price. User can add an arbitrary number of options while building strategies. The source code is available on GitHub and can be found by link below.

frontier

Built on top of pystock, this is a python module which can be used to visualize efficient frontier. The module uses Monte-Carlo simulations to generate random portfolios and then plots the efficient frontier. The source code is available on GitHub.
